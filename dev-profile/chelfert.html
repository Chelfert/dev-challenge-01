    5 topics we talked about in class

  1. What is Git? - Git is version control software
  2. What is a Git pull? - Pull requests let you tell others about changes you've pushed to a branch in a repository on GitHub.
  3. What is a Git push? - The git push command is used to upload local repository content to a remote repository. Pushing is how you transfer commits from your local repository to a remote repo.
  4. What is a Git commit? - The commit command is used to save your changes to the local repository.
  5. What is BASH? - Bash is the shell, or command language interpreter, for the GNU operating system. The name is an acronym for the ‘Bourne-Again SHell’
